## Algorithms

### Firefly Swarm Optimization

Implementation of the [firefly algorithm](https://en.wikipedia.org/wiki/Firefly_algorithm) by Yang et al. 

To run this algorithm on the DTLZ1 problem, use:

```
./gradlew run firefly
```
